管理用户
1、查询用户
所有用户信息都是存放在系统数据库mysql的user表
Use mysql;
Select * from user;
2、创建用户
通配符是%，比如用户dc，任意host =》 ‘dc’@‘%’， 可以理解为匹配任意长任意字符
Create user '用户名’@’主机名’ identified by ’密码’;
3、修改用户密码
mysql_native_password为固定内容
Alter user ‘用户名’@‘主机名’ identified with mysql_native_password by ‘新密码';
4、删除用户
Drop user ‘用户名’@‘主机名’;

权限控制

1、查询权限
Show grants for ‘用户名’@‘主机名’;
2、授予权限
任意库、任意表用*，所有库所有表*.*
Grant 权限列表 on 数据库.表名 to ’用户名’@‘主机名';
3、撤销权限
用户退出后，再次登陆才生效！
Revoke 权限列表 on 数据库.表名 from ‘用户名’@‘主机名';
