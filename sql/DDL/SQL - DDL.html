库操作

显示所有数据库
show databases;
查询当前处于哪个数据库
Select database();
创建数据库
Create database [if not exists] name;
进入、使用数据库
Use name;

表操作

查询当前数据库所有表
Show tables;
查询表结构
Desc name;
查询指定表的建表语句
Show create table name;
创建表
Create table name(
    字段1 字段1类型[comment 字段1注释],
    字段2 字段1类型[comment 字段2注释],
    字段3 字段1类型[comment 字段3注释],
    字段4 字段1类型[comment 字段4注释]       ## 注意末尾行不能有’，’
)[comment 表注释];
## 最后一句表注释不能与前一句合并写
添加字段
Alter table name add 字段名 类型(长度) [comment 注释] [约束];
修改数据类型
Alter table name modify 字段名 新数据类型(长度);
修改字段名和字段类型
Alter table name change 旧字段名 新字段名 类型(长度) [comment 注释] [约束];
删除字段
Alter table name drop 字段名;
修改表名
Alter table name rename to new name;
删除表
Drop table [if exists] name；
删除指定表，并重新创建该表(相当于清空表数据，建立了了新表)
Truncate table name;
