笛卡尔积：笛卡尔乘积是指在数学中，俩个集合a和b的所有组合情况（在多表查询时，需要消除无效的笛卡尔积）

多表查询分类
连接查询：
  • 内连接：相当于查询a，b的交集数据
  • 外链接：
  •     左连接：查询左表所有数据，以及俩表交集部分数据
  •     右连接：查询右表所有数据，以及俩表交集部分数据
  • 自连接：当前表与自身的连接查询，自连接必须使用表别名

内连接：
distinct去重

隐式内连接：
Select 字段列表 from 表1，表2 where 条件….；

显示内连接：
注意on
Select 字段列表 from 表1 [inner] join 表2 on 连接条件…;

外链接：

左外连接：
注意on
Select 字段列表 from 表1 left [outer] join 表2 on 条件…;

右外连接：
注意on
Select 字段列表 from 表1 right [outer] join 表2 on 条件…；


自连接：
语法
注意on
Select 字段列表 from 表A 别名A join 表A 别名B on 条件…；


联合查询 - union, union all:

1、对于union查询，就是把多次查询结果合并起来，形成一个新的查询结果
2、不加all是去重效果
3、查询的字段列表必须一致
Select 字段列表 from 表a…..
union [all]  
Select 字段列表 from 表b...


子查询：
嵌套查询又称自查询，外部语句可以是增删改查任意
Select * from t1 where column1 = (select column1 from t2);

根据子查询结果不同分为：
  • 标量子查询（子查询结果为单个值）
  • 列子查询（子查询结果为一列）
  • 行子查询（子查询结果为一行）
  • 表子查询（子查询结果为多行多列）

根据子查询的位置分为：
  • select后子查询
  • where后子查询
  • from后子查询


标量子查询常用操作符：=  != > < >= <=
列子查询常用操作符：in not in any(some) all
行子查询返回的结果是一行（可以是多列）: = <> in not in
Select * from table where (字段1，字段2，字段3) 操作符 (行子查询语句，结果行列数一定要与需求相同)
表子查询常用操作符: in

# 1、查询员工的姓名，年龄、职业、部门信息

# 隐式内链方式， where条件去掉无效笛卡尔积
select e.name as '姓名', e.age as '年龄', e.job as '职业', d.name as '部门'
from employee e,
     dept d
where e.dept_id = d.id;

# 显示内链
select e.name as '姓名', e.age as '年龄', e.job as '职业', d.name as '部门'
from employee e inner join dept d
    on e.dept_id = d.id;

# 左外方式
# 问题：因为是左联，所以左表内全部数据都会被搜索到，结果是不准确到
select e.name as '姓名', e.age as '年龄', e.job as '职业', d.name as '部门'
from employee e left join dept d on e.dept_id = d.id;


# 2、查询年龄小于30岁的员工姓名、年龄、职位、部门信息

# 自己使用子查询 + 左联
select e.name as '姓名', e.age as '年龄', e.job as '职业', d.name as '部门'
from (select * from employee where age < 30) as e left join dept d on e.dept_id = d.id;

# 隐式内链
select e.name as '姓名', e.age as '年龄', e.job as '职业', d.name as '部门'
from employee e, dept d
where e.age < 30 and e.dept_id = d.id;

# 显示内链
select e.name as '姓名', e.age as '年龄', e.job as '职业', d.name as '部门'
from employee e inner join dept d on e.dept_id = d.id and e.age < 30;


# 3、查询拥有员工的部门id，部门名称

select d.id '部门id', d.name '部门名称'
from (select * from dept) d where d.id in (select dept_id from employee);

# 使用内连接实现，内连接：获取的就是交集部分，即员工表中无部门id的数据不取，部门表无员工的数据不取
# 注意取出无效笛卡尔积
# 注意去重数据
select distinct d.id '部门id', d.name '部门名称' from employee e, dept d where e.dept_id = d.id;


# 4、查询所有年龄大于40岁的员工，及其归属的部门名称；如果员工没有分配部门，也需要展示出来
select e.*, d.name '部门' from (select * from employee where age > 40) e left join dept d on e.dept_id = d.id;


# 5、查询所有的员工工资等级

select e.name '姓名', s.grade '工资等级'
from (select name, salary from employee) e
    join salgrade s on e.salary >= s.losal and e.salary <= s.hisal;


# 6、查询“研发部”所有员工的信息及工资等级
select e.*, s.grade '工资等级'
from (select * from employee where dept_id = (select id from dept where name = '研发部')) e
         join salgrade s on e.salary >= s.losal and e.salary <= s.hisal;

# 7、查询“研发部”员工的平均工资
select avg(e.salary) '研发部平均工资'
from employee e
where e.dept_id = (select id from dept where name = '研发部');

# 8、查询工资比“灭绝”高的员工信息
select * from employee where salary > (select salary from employee where name = '灭绝');

# 9、查询比平均薪资高的员工信息
select * from employee where salary > (select avg(salary) from employee);

# 10、查询低于本部门平均工资的员工信息
select * from employee e
    join (select dept_id did, avg(salary) savg from employee group by dept_id) s
         where e.dept_id = did and e.salary < savg;

# 11、查询所有的部门信息，并统计部门员工的人数
select d.name '部门', count(*) '人数' from employee e join dept d on e.dept_id = d.id group by dept_id;
